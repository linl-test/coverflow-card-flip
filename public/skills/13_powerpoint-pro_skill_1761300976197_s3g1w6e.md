# PowerPoint Pro

- Author: Anthropic
- Updated: 2026-01-09T05:57:03.401Z

## About this skill

Professional PowerPoint presentation toolkit for creating, editing, and analyzing .pptx files. Create stunning presentations from scratch using html2pptx workflow that converts HTML slides to PowerPoint with pixel-perfect positioning, supporting charts, tables, and complex layouts. Edit existing presentations through OOXML manipulation with validation at each step. Work with templates by duplicating, rearranging, and replacing content using inventory-based text replacement with proper formatting preservation. Apply sophisticated design principles including content-informed color palettes, creative typography treatments, and professional layouts. Generate thumbnail grids for visual analysis, convert slides to images, and handle speaker notes, animations, and slide masters. Includes 18 curated color palettes and extensive design guidelines.
