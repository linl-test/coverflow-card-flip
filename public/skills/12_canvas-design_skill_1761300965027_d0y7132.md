# Canvas Design

- Author: Anthropic
- Updated: 2026-01-09T05:57:00.610Z

## About this skill

Create museum-quality visual art in PNG and PDF formats using sophisticated design philosophy principles. Follows a two-step creative process: first articulating a visual philosophy (aesthetic movement expressed through form, space, color, composition), then expressing it through masterfully crafted artwork. Emphasizes visual communication over text, treating words as rare visual accents integrated into the design. Supports multiple output formats, sophisticated typography from the canvas-fonts directory, and can create multi-page series. Perfect for creating abstract posters, conceptual art pieces, exhibition materials, book covers, and sophisticated visual compositions that combine artistic intuition with rigorous design principles.
